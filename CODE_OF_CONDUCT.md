# Contributor Covenant Code of Conduct

## Our Pledge
We as contributors and maintainers pledge to make participation in this project
a harassment-free experience for everyone.

## Our Standards
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Gracefully accepting constructive criticism
- Focusing on what is best for the community

## Enforcement
Instances of abusive or unacceptable behavior may be reported by opening an issue
or contacting the maintainer directly via GitHub.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
